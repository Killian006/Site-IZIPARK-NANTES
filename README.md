# ✈️ Izipark Nantes - Refonte Conceptuelle (Homepage)

Ce projet est une proposition de refonte (non-officielle) de la page d'accueil du site de réservation de parking **Izipark Nantes**. 

L'objectif de ce prototype est de démontrer comment une modernisation de l'interface utilisateur (UI) et de l'expérience utilisateur (UX) peut fluidifier le parcours client et potentiellement augmenter le taux de conversion, tout en conservant les logiques métier existantes.

👉 **[Voir le site en direct (Live Demo)](https://killian006.github.io/Site-IZIPARK-NANTES/)**

## ✨ Fonctionnalités et Améliorations

* **Design Moderne & Responsive :** Utilisation de CSS Grid et Flexbox pour garantir un affichage parfait sur tous les supports (Mobiles, Tablettes, Desktop).
* **Internationalisation (i18n) native :** Mise en place d'un script JavaScript permettant de basculer l'ensemble des textes du Français à l'Anglais de manière instantanée, sans rechargement de page.
* **Animations & Micro-interactions :** * Preloader personnalisé (animation SVG).
  * Effets d'apparition fluides au défilement (Intersection Observer API).
  * Défilement doux (Smooth scrolling) pour les ancres.
* **Respect de la logique métier :** Le module de réservation a été redésigné mais conserve sa compatibilité avec l'API d'origine (*Inkara*). Un script reformate la volée les dates et heures saisies par l'utilisateur pour qu'elles soient comprises par le serveur.
* **Performances & SEO :** Code allégé, suppression des balises de style en ligne, structure HTML5 sémantique.

## 🛠️ Stack Technique

* **HTML5** (Sémantique et accessible)
* **CSS3** (Variables natives, Grid, Flexbox, Animations)
* **JavaScript (Vanilla)** pour la logique d'UI (i18n, Scroll reveal, Preloader)
* Intégration conservée de bibliothèques tierces (jQuery, Pickmeup.js pour le calendrier, Tarteaucitron pour le RGPD).
